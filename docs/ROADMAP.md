# Roadmap

## MVP

- Local encrypted vaults.
- CLI lifecycle and item commands.
- Encrypted backup import/export.
- CSV import.
- Dioxus UI surfaces.
- Self-hosted sync API and Docker deployment.

## Next

- Full desktop shell integration.
- Browser-friendly web storage adapter.
- Multi-device sync client configuration.
- Optional plaintext export behind an explicit dangerous flag.
- Recovery-code workflows and richer duplicate-resolution UI.

## Later

- Team sharing using public-key recipient wrapping.
- Hardware-backed key support.
- Mobile clients.
- External password-manager importers.
